多项式乘法

前置知识

系数表示法

$f(x)=\sum_{i=0}^{n} a_ix^i$可以描述一个$n$次多项式。

点值表示法

众所周知，$n+1$个点可以唯一地确定一个$n$次多项式。

例如给你两点$(0,0)$，$(1,1)$，可以确定一次函数$y=x$

故用$(x_0, y_0), \cdots, (x_n, y_n)$表示$f(x)$也是一个方法。

复数

形如$a+bi$，其中$i^2=-1$，在复平面（x实轴，y虚轴）上表示从$(0,0)$到$(a,b)$的向量。

四则运算：

模长：$(a,b)$到$(0,0)$的距离

幅角：从正半轴逆时针转到向量的角度

乘法的几何意义：模长相乘，幅角相加

  • 例题：计算$\frac{2+i}{2-i}$

单位根

在复平面上以原点为圆心作一个半径为1的圆，并将其$n$等分，将每条$n$等分的边看做一条向量。（图为$n=8$的情形）

记$w_n$表示幅角为正且最小的向量所代表的复数，在图中表示为从正半轴逆时针数一个的向量。（即朝右上方的向量）

那么从正半轴开始，逆时针数，每个向量表示的复数分别是：$w_n^0, w_n^1, \cdots, w_n^{n-1}$，这是由于乘法的几何意义：模长相乘，幅角相加，乘上一个$w_n$相当于逆时针旋转$\frac{360 ^{\circ}}{n}$，模长不变。

显然$w_n^0=w_n^n=1$。

代数意义：$z^n=1$的所有解。

快速傅里叶变换(FFT)

思想

快速傅里叶变换(FFT)的大体框架

也即$\text{FFT(Fast Fourier Transform)}$，可以$O(n \log n)$地计算两个多项式$F(x)$和$G(x)$的卷积（这里可以理解为乘积）

其思路如下：

将$F(x)$和$G(x)$从系数表示$O(n\log n)$地转换为点值表示（我们需要两倍长度的点）

然后将所有$y_i$都乘起来（这一步是$O(n)$的，容易实现）

再$O(n\log n)$地转换为系数表示，得到$F(x)G(x)$

现在我们有问题：选取哪些$x_i$？这会在下文给出解答。

  • 例题：给$y=x$和$y=2x+1$，通过上述方法（系数表示->点值表示->系数表示）算出乘积。

离散型傅里叶逆变换(IDFT)

即$\text{Inverse Discrete Fourier Transform}$。

可以证明：

对某一多项式$A(x)=a_0x^0+a_1x^1+\cdots+a_nx^n$

设$B(x)$的每一项系数$b_i$为$A(w_n^i)$

且$C(x)$的每一项系数$c_i$为$B(w_n^{-i})$

则有$a_i=\frac{c_i}{n}$

证明过于繁琐，如果想知道可以翻网上别的解法。

于是当我们求出$F(x)G(x)$在$w_n^0,\cdots,w_n^{n-1}$处的点值表达时，把点值表达串成一个多项式做IDFT，求出在$w_n^0,w_n^{-1},\cdots,w_n^{-(n-1)}$处的系数，最后将系数除以$n$就能得到$F(x)G(x)$的系数表达。

快速数论变换(NTT)

快读数论变换(NTT)的大体框架

也即$\text{Number Theoretic Transform}$，求$F(x)G(x) \mod p$的一类算法。

事实上大体框架与FFT完全相同，不同之处在于用原根替换掉单位根。

一般来说，$p=998244353$，它的原根$g=3$。

原理

我们需要让素数$p$满足$p=k\times 2^m+1$，（事实上，$998244353-1=119\times2^{23}$）设其原根为$g$

令$g_n=g^{\frac{p-1}{n}}$。（当我们做NTT时，n被补到2的幂次，于是就可以除）

考虑为什么可以使用单位根优化DFT，根本原因是单位根满足一些性质：

  • $w_n^k$两两不同$(0\le k<n)$ （足够确定多项式）

我们可以证明

假设存在$g_n^i\equiv g_n^j \pmod p (i>j)$

有$g_n^{i-j} \equiv 1 \pmod p$

而$i-j<\varphi(p)$，于是我们找到一个比$\varphi(p)$小的$t$满足$g^t\equiv 1 \pmod p$这与$g$是原根的定义矛盾。

而在NTT的过程中不会出现$g_n^{n-1}$，所以可以保证所有$x_i$都是不同的。

  • $w^{dk}_{dn}=w^k_n$ （可以分成子问题）

我们有$g_{dn}^{dk}\equiv g^{\frac{p-1}{dn}\times dk}\equiv g_n^k$

所以用原根替换单位根是可行的。

多项式求逆

问题描述

给定$n$次多项式$F(x)$，求一多项式$G(x)$，使得

$G(x)$的系数$\pmod {998244353}$（或者一个别的模数？）

luogu 多项式求逆

基础结论

假设我们得到了一个$\frac{n}{2}$次多项式$H(x)$满足$F(x)H(x)\equiv 1\pmod {x^{\frac{n}{2}}}$

由于$F(x)G(x)\equiv 1\pmod {x^{\frac{n}{2}}}$，所以$(G(x)-H(x))\equiv 0 \pmod {x^{\frac{n}{2}}}$

令$A(x)=G(x)-H(x)$，我们可以发现$a_0,a_1,\cdots,a_{\frac{n}{2}-1}$都应为$0$。

根据直觉或者将整数乘法归纳到多项式乘法上$A^2(x)$的系数$t_i=\sum_{k=0}^{i} a_ka_{i-k}$，考虑到对$i<n$，$\min\{k,i-k\}<\frac{n}{2}$，所以$A^2(x)\equiv 0 \pmod {x^n}$

那么$G^2(x)-2G(x)H(x)+H^2(x)\equiv 0 \pmod {x^n}$，同时乘上$F(x)$，并利用$G(x)F(x)\equiv 1 \pmod {x^n}$，得出$G(x)\equiv 2H(x)-F(x)H^2(x)$.

而我们发现如果我们要求$H(x)$，把$H(x)$当成$G(x)$，这便是一个子问题了！

现在我们算复杂度，$T(n)=O(n\log n)+T(\frac{n}{2})$，由于master定理似乎无法解决，我们考虑乱搞。

所以差不多是$O(n\log n)$的复杂度…

细节

观察以上求逆元的代码，需要注意：

  • 在使用的最开始，需要补齐$F(x)$到2的幂次
  • $2H(x)-F(x)H^2(x)$是一个$2n$次的式子，不能因为其$\pmod {x^n}$就只将NTT的长度设为$n$。
  • 那么由于这个式子是$2n$次的，IDFT后要将多余的$(n+1)$~$(2n)$这些都设为$0$
  • 边界条件：长度为1时，$G_0=F_0^{-1}$
