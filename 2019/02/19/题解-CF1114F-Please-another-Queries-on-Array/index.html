题解 CF1114F Please, another Queries on Array?

发表于 2019-02-19 | 分类于 OI |

题面

CF1114F

题意

给一序列$\{a\}$，要求支持两个操作：

  1. MULTIPLY l r x 对每个$l\le i\le r$执行$a_i\leftarrow a_i\times x$
  2. TOTIENT l r 询问$\varphi(\prod_{i=l}^{r} a_i) \mod 10^9+7$

$1\le n\le 4\times 10^5, 1\le q\le 2\times 10^5, 1\le a_i\le 300, 1\le x\le 300$

解法

一开始实际上是想用线段树维护区间乘积，然后每次询问的时候计算欧拉函数，但这样区间乘积就不能模，就很麻烦。

考虑把这个欧拉函数拆开，设质因数分解式$\prod_{i=l}^{r} a_i=\prod_{j=1}^{m} p_j^{k_j}$，$p_j$是质数，$k_j\ge 1$

所以我们需要维护两个东西：区间乘积，区间含质因子集合。

注意到值域在$[1,300]$，经过计算，其中的质数有62个，如果我们用一long long的每一二进制位来存质数集合，刚好存一个long long。

而区间乘积，区间含质因子集合显然具有加性（维护集合时将两个子区间代表集合的long long数做或运算），于是使用线段树维护，询问时需要用到逆元。

$O((n+q)\log n))$.

总结

实际上都比较套路，但我在考试时就不容易想到可以用一long long数来存储质因子，我在考试时只想到$O(P(n+q)\log n)$的解法(P=62)，而且没有尝试去写，因为两棵线段树在两小时写完还是比较有难度。
